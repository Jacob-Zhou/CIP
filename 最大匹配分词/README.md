# 最大匹配分词

## 一.目录结构

```
./data/:
    data.conll: 数据文件
    data.txt: 生成的毛文本文件
    word.dict: 生成的词典文件
    data_MM.out: 生成的前向最大匹配分词结果文件
    data_RMM.out: 生成的后向最大匹配分词结果文件
max_match.py: 最大匹配分词代码
max_match.ppt: 参考ppt
./README.md: 使用说明
```



## 二.运行

### 1.运行环境

```
python 3
```

### 2.运行方法

```bash
python max_match.py
```
